# DataDogTest

A simple app that demonstrates how to integrate DataDog (1.x) automatic network request instrumentation with Ktor on iOS.


## KMP Dependencies

* Ktor 2.3.6
* KMP-NativeCoroutines 1.0.0-ALPHA-20
* Koin 3.5.0

## iOS Dependencies

* dd-sdk-ios 1.17.0
* KMP-NativeCoroutines 1.0.0-ALPHA-20
